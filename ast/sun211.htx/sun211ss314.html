IsLatAxis(axis)

Is the specified celestial axis a latitude axis?

Description:

This is a read-only boolean attribute that indicates the nature of the specified axis. The attribute has a non-zero value if the specified axis is a celestial latitude axis (Declination, Galactic latitude, etc), and is zero otherwise.
Type:
Integer (boolean), read-only.

Applicability

SkyFrame
All SkyFrames have this attribute.

Notes:

  • When specifying this attribute by name, it should be subscripted with the number of the SkyFrame axis to be tested.
