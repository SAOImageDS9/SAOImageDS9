SAOImageDS9 Version 8.8 Release Notes

  1. BUILD: Updated Makefiles to ensure that tclsh is getting picked up from the local build directory.
  2. OPENSSL: Updated to 3.6.2
  3. TLS: Upgraded to 2.0
    • Needed to modify several files from the upstream repro to work with DS9
  4. CERTIFI: Added certifi package to obtain SSL certificates.
    • Users can override with standard SSL environment variables
  5. HTTP: Updated to support changes in TLS 2.0
  6. WINDOWS: Changed packaging to use Inno Setup installer.
  7. CUBE: Fix problem trying to access specific cube slices (123, 132, 213, etc.)
  8. VIEW: Advanced view now supports separate icons that are dark mode friendly.
  9. THEMES: Updated latest scid themes and added dark mode options.
  10. FOOTPRINT: Updated the URL to the CXC footprint service
  11. VOTABLE: Updated to support both POST and GET requests required by CXC footprint service.
  12. ILLUSTRATE: Fix problem loading illustrate mode files (.seg) that contain a line shape.
  13. COLORBAR: Updated URL for CPT-CITY color map repository.
  14. HTTP: Updated to respect Content-Disposition header keys which contain filename value.
  15. GRAPH: Parser updated to require horizontal or vertical options
  16. DATA: Updated to return an error message if there are no frames.
  17. DATA: Updated to return an error message if there is a frame but no data loaded in it.
  18. DSSSAO: The DSS server at SAO is no longer being actively maintained, so dsssao now returns an error message.
  19. EXAMINE: Fix bug when event file is loaded, data are now properly rebinned to 1.0/zoom level.
