SAOImageDS9 Version 8.4 Release Notes

  1. 02.22.2022 TCL/TK: upgrade to version 8.6.12
  2. 02.24.2022 WCS: fixed WCS save command to work when the wcs dialog is not displayed.
  3. 02.25.2022 WCS: fixed issue where displayed WCS was no updated when blocked.
  4. 03.01.2022 WCS: insert new lines in saved wcs files.
  5. 03.01.2022 GUI: set fonts after prefs, commandline, and backup restore processed.
  6. 03.01.2022 VIEW: fixed an issue in certain cases in vertial view mode where a large font size caused an window reconfigure event loop.
  7. 03.01.2022 BACKUP: fixed an issue where older backup scripts do not keep track of the number of frames created.
  8. 03.02.2022 BACKUP: preserve window size in backup file for version number 8.4 or greater. Older back files default to startup window size.
  9. 03.02.2022 MACOS: fixed a problem with mousewheel when cursor over colorbar.
  10. 03.04.2022 IRAF: fixed a problem with imexam and loading fits files (including mosaics) into ds9 directly.
  11. 03.09.2022 FITSY: rice axis size need not be integer multiple of tile size.
  12. 03.25.2022 3D: no graphics in magnifier.
  13. 04.01.2022 PRINT: removed default setting 'Screen' as it nolonger meaning full across platforms.
  14. 04.01.2022 SCALE: fixed a minor issue with scale dialog when max-min=0.
  15. 04.01.2022 1D: enable special case: NAXES=1, NAXIS1=n.
  16. 04.04.2022 3D: fixed an issue with reorder axes of data cube.
  17. 04.05.2022 SCALE: add dialog to enter scale mode value.
  18. 04.06.2022 ANALYSIS: add combobox parameter type.
  19. 04.08.2022 RELEASE version 8.4b1
