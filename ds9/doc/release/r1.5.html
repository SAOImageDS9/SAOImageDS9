SAOImageDS9 Version 1.5 Release Notes

Version 1.5

  1. Fixed problem with Windows style Open file dialog box
  2. Now supports following coords systems:
    • Image (was Logical)
    • CCD (was Physical)
    • Detector (was Mosaic Physical)
    • Amplifier
    • WCS
  4. New version of Doug Mink's WCS libs
  5. Fixed minor problems with interactive Pan Mode
  6. Added "Source TCL" file support
  7. Open Mosaic Image command

  8. Will open multiple fits files as a mosaic.
    Supports following patterns:
    *#.fits
    *_#.fits
    *.IM*.fits
  9. Open Mosaic Segment command
  10. Fixed minor problem with zscale and high energy data
  11. The Magnifier now Works!

  12. There are two algorithms available (under Preferences)
    Magnification can be selected via preferences
    1. Fast (and less Accurate)
    2. More Accurate (and slower)
  13. Improvements to Dialogs
  14. Now supports (unoffical)FITS data size -16 (unsigned short)
  15. Support for byte swapped processors
  16. Support large images > 8k x 8k
  17. Loading images are now 10x faster
  18. Supports BLANK Fits keyword
  19.  Menu Short Cuts have been added (use Alt key, such as Alt-f for Frame menu)
  20. Horizontal and Verticle Graphs now work in all modes (not just Crosshair)
  21. ds9 will accept fits files via STDIN (ie. $ cat foo.fits | ds9 -file -  )
  22. command line argument support (ie. $ds9 -? for list)
  23. crosshair position is more accurate
  24. added new scales: 'ln' and 'log'.
    1. ln --  SAOimage natural log based algorithm
    2. log--  IRAF log base 10 algorithm
  26. Support for Postscript printing (Level 1 and Level 2)
  27. Blinking is now supported
